## 작업 내용

### 1. ML 모델 개발

#### 1.1 데이터 준비
**파일**: `backend/app/ml/data_preparation.py`

**작업 내용**:
- 시세 데이터 전처리
- 특징(Feature) 추출
- 라벨링 (매수/매도/보유)
- 데이터 정규화
- 학습/검증/테스트 세트 분할

**특징 예시**:
- 기술적 지표 (RSI, MACD, 이동평균 등)
- 가격 변동률
- 거래량 지표
- 시장 지표 (코스피 지수 등)

#### 1.2 모델 선택 및 학습
**파일**: `backend/app/ml/models.py`

**모델 옵션**:
- 시계열 예측: LSTM, GRU, Transformer
- 분류: Random Forest, XGBoost, LightGBM
- 강화학습: DQN, PPO (선택)

**학습 프로세스**:
1. 모델 아키텍처 정의
2. 하이퍼파라미터 튜닝
3. 모델 학습
4. 검증 세트로 평가
5. 모델 저장

#### 1.3 모델 평가 및 최적화
**파일**: `backend/app/ml/evaluation.py`

**평가 지표**:
- 정확도 (Accuracy)
- 정밀도 (Precision)
- 재현율 (Recall)
- F1 Score
- 백테스팅 성과

### 2. 전략 통합

#### 2.1 ML 전략 인터페이스 구현
**파일**: `backend/app/trading/strategy/ml_strategy.py`

**구현 내용**:
- `Strategy` 인터페이스 구현
- ML 모델 로드
- 실시간 예측
- 신호 생성

#### 2.2 전략 레지스트리 등록
**파일**: `backend/app/trading/strategy/registry.py`

**작업**:
- ML 전략 등록
- 전략 설정 관리
- 모델 버전 관리

#### 2.3 프론트엔드 통합
**컴포넌트**: `frontend/components/trading/MLStrategyConfig.tsx`

**기능**:
- ML 전략 선택 UI
- 모델 파라미터 설정
- 학습 데이터 기간 설정
- 모델 재학습 트리거

### 3. 성능 최적화

#### 3.1 모델 최적화
**작업 내용**:
